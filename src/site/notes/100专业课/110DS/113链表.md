# 1 单链表 
## 1.1 单链表的结点定义
- LinkList = LNode *
```C
LinkList L;		//侧重定义链表
LNode *p;		//侧重定义结点
```
- 头指针通常表示：head、L
- 如何表示单链表：定义一个头指针 L 并将其指向单链表的第一个结点
- 单链表结点的**规范定义**：
```C
#include<stdio.h>
//1.先将数据域单独定义为一个结构类型
typedef struct{
    char num[8];    //数据域
    char name[8];   //数据域
    int score;      //数据域
}ElemType;
//2.直接用刚定义的结构类型来定义结点类型的数据域
typedef struct Lnode{
    ElemType data;       //数据域
    struct Lnode *next;  //指针域
}Lnode, *LinkList;
```
